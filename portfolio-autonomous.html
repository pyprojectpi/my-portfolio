Autonomous Vehicle using Raspberry Pi

Designed and developed a self-driving mini car powered by Raspberry Pi. The system uses computer vision techniques such as lane detection, object tracking, and obstacle avoidance to navigate autonomously. The project demonstrates the integration of real-time image processing, motor control, and decision-making.

This low-cost prototype helps simulate intelligent transport systems and encourages safer, smarter mobility solutions using embedded AI.
