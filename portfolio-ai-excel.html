AI Excel Mock Interviewer

Project Overview

The AI Excel Mock Interviewer is an interactive web application designed to help users practice Excel interview questions. It leverages the Gemini 2.0 API to evaluate answers, provide feedback, and simulate a real interview environment.

Key Features:

  • AI-driven question and answer evaluation
  • Real-time candidate feedback
  • Supports common Excel functions, formulas, and scenarios
  • Web-based, interactive, and responsive interface
