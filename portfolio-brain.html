Brain Tumor Detection using YOLOv12

This project focuses on detecting brain tumors in MRI scans using the YOLOv12 deep learning model. The system achieves fast and accurate localization of tumor regions, making it suitable for real-time medical screening support. Image preprocessing, annotation, model training, and evaluation were performed using Python and OpenCV.

The goal is to assist healthcare professionals by improving early diagnosis, increasing efficiency, and reducing manual error in interpreting MRI images.
